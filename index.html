What is Tailwind CSS?

  • Utility first CSS framework
  • Highly customisable
  • More than a CSS framework, rather an engine for creation of design systems

Example Values

  • Text/border/background colors
  • Font-size/family-weight
  • Padding/margin/negative margin
  • Flexbox
  • Positioning
  • Lists
  • z-index
  • Opacity

Example Values

  • Screenreader visibility
  • Placeholder color
  • first-child, last-child, nth-child
  • CSS Grid
  • Transition
  • Transform
  • Spacing/Divide

How do I use it?

  • Style elements by applying pre-existing classes directly in HTML
  • Use utility classes to build designs without writing CSS

What are the benefits?

  • Not wasting time on inventing class names
  • The CSS stops growing
  • Easier to change things
  • Easier to identify the cause of issues due to clear naming of classes

How can I install Tailwind?

  • 1. Use the CDN
  • 2. Installing it via NPM

Installing via NPM

                            npm install --save-dev tailwindcss

yarn add -D tailwindcss

Including Tailwind in CSS

                            /* src/css/tailwind.pcss */

@tailwind base;

@tailwind components;

@tailwind utilities;

Adding your own classes

                            /* src/css/tailwind.pcss */

@tailwind base;
/* Custom base styles */

@tailwind components;
/* Custom components */

@tailwind utilities;
/* Custom utilities */

Adding your own classes (with layers)

                            /* src/css/tailwind.pcss */

@tailwind base;
@tailwind components;
@tailwind utilities;

@layer components {
 /* Custom components */
}

Processing your CSS with build command

                            npx tailwind build

src/css/tailwind.pcss

-o dist/app.css

General structure for interaction classes

[state][seperator][class]

Interaction classes example

hover:text-red-500

focus:text-red-500

Interaction states in CSS

                            .text-red-500 {
 color: #f56565;
}

.hover\:text-red-500:hover {
 color: #f56565;
}

.focus\:text-red-500:focus {
